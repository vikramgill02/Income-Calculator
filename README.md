# Income-Calculator

Single-page app deployed at <https://vikramgill02.github.io/Income-Calculator/>. Two tabs:

- **Pay Calc** — estimate a future paycheck from days/hours at Sutter and NorthBay.
- **YTD** — year-to-date gross income, broken down by employer, plus running 403(b) total. Data lives in [`paystubs.json`](./paystubs.json).
